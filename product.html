About the Product

What is the product?

Our Face Recognition Attendance System uses Raspberry Pi, OpenCV, and deep learning libraries to replace manual roll calls. It captures a student’s face, matches it against the database, and logs their attendance instantly.

Why is it useful?

Attendance is stored in a secure database and can be exported to Excel. It saves teachers’ time, reduces human error, and provides a scalable solution for classrooms and institutions.

How it works

1. Students’ photos are captured and stored in a dataset.
2. The system trains a recognition model.
3. On class day, the camera recognizes faces in real time.
4. Confirmation is asked before logging into the attendance database.
5. LED lights give instant feedback (green for success, red for reject).

Key Features

- Real-time face recognition
- SQLite + Excel export
- LED feedback system
- Eco-friendly recycled casing
- Affordable and scalable design
