Our Intelligent Security System



What is Neo SynthX?

Our innovative system uses a combination of Face Recognition, RFID, and Raspberry Pi technology to create a secure, automated access control solution. It effectively replaces outdated sign-in sheets or manual checks, offering a reliable, modern, and efficient way to manage entry to sensitive areas.

See it in Action

Secure & Instant Logging

Every successful or failed access attempt is stored in a secure, encrypted database. This eliminates human error and provides an instant, traceable record. Logs can be easily accessed via the web dashboard, allowing management to generate reports and track access patterns efficiently.

  • Increased Security: Role-based access control (Worker, Manager, CEO).
  • Reduced Admin Time: Automated logging means less paperwork.
  • Reliable Data: Accurate, timestamped records for auditing.

How the System Works

The system operates through a simple, yet robust sequence of steps for every attempted entry:

  • Enrollment: User photos and RFID tags are registered and encrypted.
  • Recognition: The camera captures the face and matches it against the trained model.
  • Verification: The RFID tag provides a secondary layer of authentication.
  • Access Control: A real-time check of user permissions determines access rights.
  • Feedback: An OLED display shows immediate status (Access Granted/Denied).
  • Logging: All events are instantly recorded to the central database.
