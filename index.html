Introduction

Our New Intelligent Security Access System

Our system uses Face Recognition + RFID + Raspberry Pi to provide secure, automated access control. Each user is assigned a role (Worker, Manager, CEO) and entry is granted or denied based on their permissions. An OLED display shows real-time access status, and all access logs are stored in a central web dashboard.
